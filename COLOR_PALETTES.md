# Premium Color Palettes for Photography Booking Platform

## Option 1: Elegant Noir & Gold (Luxury/High-End)
**Best for:** High-end wedding photography, luxury portrait studios
- **Primary:** `#D4AF37` (Gold) - CTAs, highlights, premium badges
- **Secondary:** `#1A1A1A` (Deep Black) - Text, headers, footer
- **Accent:** `#F5F5F5` (Soft White) - Backgrounds, cards
- **Highlight:** `#8B7355` (Bronze) - Hover states, borders

```css
/* Usage Example */
background: linear-gradient(135deg, #D4AF37 0%, #8B7355 100%);
```
